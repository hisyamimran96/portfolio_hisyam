Overview

The project can be categorised into stages:

  1. Database Design - design the relations between tables. In this stage I determined the fact and dimension tables of the whole database.

  2. Data Extraction - create connection between source and destination. In this case I will be using an API endpoint from Alpaca as my main primary source of data.

  3. Data Cleaning/transformation - Clean the data, this includes dropping columns that are not useful for our use. For this project, I will only need the asset name, the asset ticker symbol and the what exchange the asset is listed in. Also, the data will be filtered into a format that is easier for us to handle, this will be dicussed further below.

  1. Data Loading - load the data into locally hosted PostgreSQL database.

Tools used in this project:

  1. Python - our main programming language.

  2. Docker - to streamline development environments. So we can forget about dependencies problem with different OS.

  3. TimescaleDB - Provide time-series and analytics functionality on-top of PostgreSQL. It supports the use of SQL for querying. Basically a RDBMS for time-series. This can be used further down when we want to analyse more complex data-sets like stock prices etc. but for now we are not using this yet.

  4. PostgreSQL - for database management - CRUD operations. We use postgres to store stocks data.

  5. TablePlus - GUI for interacting with postgreSQL. (optional, we can still use terminal to execute queries)

Data Extraction

We will establish a connection with Alpaca’s API endpoint to extract market data. Read here for more info on Alpaca API. First install and import the required package into our environment.

import alpaca_trade_api as alpaca

With reference to the Alpaca API docs, we establish the endpoint connection using the following code:

api = alpaca.REST(config.api_key, config.api_secret, base_url = config.endpoint)

And finally we can extract all market data using the list_assets() method:

assets = api.list_assets()

When running this method, python will fetch all the market data. But we want to do some cleaning, only extracting what we want.
